# Game Collection Website
This project is a practice website that showcases games I have played. The application allows you to explore games and recommend new games for me to try

## Home Page

The main page displays two random games from my collection. The games change randomly every 3 seconds, giving a quick preview of different games

## All Games Page

This page contains a complete list of all games I have played. You can browse the collection and search for games

## Recommendation Page
You can recommend a new game for me to try
When you submit a recommendation through the form, the suggestion is sent directly to my email

## Technologies Used

- Next.js – used to build the full-stack web application
- React – used for building the user interface
- TypeScript – used to add static typing and improve code reliability
- MongoDB – used as the database to store all game information
- Tailwind CSS – used for styling the application
- Nodemailer – used to send game recommendations to my email
